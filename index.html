Python-class(类)的用法

2018-01-09

俗话说.Python-里面一切皆对象.变量.字符.函数.类.而面向对象有三大特性.封装.继承.多态.对于一个完全面向对象的语言来说.Python-类(class)的实现更简洁和方便.类的使用是: 类定义用.class–> 类对象—>实例对象.面向对象的语言有一个特点是.圆点的天下

封装:将数据整合到一起.就好比将杂乱无序的放到一个空的盒子里面去
继承:减少代码的重复编写.子承父业一样.儿子不用努力就可以得到一切
多态:可以随意的为对象添加和修改类型或者属性

class Date(object):# 用 class 来定义一个类对象.这个类对象继承于 object。这个 object 是 Python 里面定义号的一个类.用来可以实现那些方法
    """类的创建"""
    pass

if __name__ == "__main__":
    date = Date()# 由类对象来创建实例对象.一个类对象可以创造出 N 个实例对象来
    date2 = Date()
    date3 = Date()

class Attrbute(object):
    """类的属性"""
    name = '你是一个大笨蛋'# 创建一个类属性.类对象和实例对象都可以进行访问

if __name__ == "__main__":
    att = Attrbute()

    print(Attrbute.name)# 类对象调用类属性

class Priv(object):
    """类的私有属性"""
    __age = 999# Python 没有向其他语言定义私有变量和公有变量.Python 的做法的定义私有的属性在这个属性前面加上两个下划线就行.使外部程序无法直接的调用

    def return_age(self):
        """实例方法"""
        return Priv.__age

if __name__ == "__main__":

Python 分支和循环列表

|    分支                说明
|----------------------------------------------------
|    if                 当条件为 True(真) 时执行下面的语句
|----------------------------------------------------
|    elif             当 if 条件为 False(假) 时执行.比须在 if 条件下面
|-----------------------------------------------------------------
|    else             当 if 或 elif 都为假时.执行。比须在 if 条件下面
|-----------------------------------------------------------------
|    循环                说明
|-----------------------------------------------------------------

|    while              条件为 True(真).时就入循环体.为假则不执行
